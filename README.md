# The Power of Life

Sitio web construido con Next.js + Tailwind CSS. Contiene páginas estáticas en `src/app/` y componentes reutilizables en `src/components/`.

Quick start

1. Instala dependencias:

```bash
npm install
```

2. En desarrollo:

```bash
npm run dev
```

3. Build de producción:

```bash
npm run build
npm run start
```

CI

Incluí un workflow básico que corre `npm ci` y `npm run build` en cada push/PR para validar la compilación.

Remoto

El repositorio remoto configurado es: `https://github.com/thewonderhubteam-demos/the-power-of-life.git`.

Notas
